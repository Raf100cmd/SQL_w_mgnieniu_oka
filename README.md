# SQL_w_mgnieniu_oka

W książce o tożsamym tytule jak repozytorium są przedstawione przeze mnie propozycje rozwiązań zadań w oparciu o przykładowe bazy danych zawarte w publikacji.
Są to następujące bazy danych:
- Klienci;
- Zamowienia;
- ElementyZamownienia;
- Produkty;
- Dostawcy.
